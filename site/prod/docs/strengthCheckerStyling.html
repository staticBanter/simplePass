strengthCheckerStyling

An interface describing the 'Strength Checker Styling' object.

Properties:
NameTypeDescription
styleTargetHTMLElement

The HTML element to style.

styleTypestring

The type of styling to use on the target. Either inline or css. If set to inline simplePass will attempt to insert inline style attributes to the given target; If set to css simplePass will attempt to add a class and data attribute to the given target, you may then use custom css to style this however you like.
