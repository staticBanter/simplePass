Interface: messageTypes

messageTypes

An interface describing the 'Message Types' object.

Properties:
NameTypeAttributesDescription
htmlMessageobject<optional>

An object used to describe the HTML message

consoleMessageboolean | object<optional>

A boolean used to indicate if the message should also be displayed in the console. Can also be an object used to signal clearing of console messages.

level"ERROR" | "WARNING" | "INFO" | "LOG" | "DEBUG" | "CLEAR"

The level the message should be displayed as to the user. Error messages will halt the program and always throw a console error message. The "CLEAR" level is used to signal clearing current messages.
