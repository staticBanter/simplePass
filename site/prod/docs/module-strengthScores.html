(inner, constant) strengthScores :object

An object that describes the strength scores assigned to a password.

Type:
  • object
Properties:
NameTypeDescription
badstrengthScoreAttributes

The weakest score a password can receive. A password with this score should not be used.

modeststrengthScoreAttributes

The second weakest score a password can receive. While passwords with this score may look secure, they often lack in entropy. Passwords that receive this score should not be used.

okstrengthScoreAttributes

This is the middle score a password can receive. This is the beginning of what we consider a 'strong' password. A password that receives this score should be considered to be updated to a 'good' password score.

goodstrengthScoreAttributes

This score is considered to be the current 'good' password strength. Passwords that receive this score are strong enough to be used as a password.

excellentstrengthScoreAttributes

The highest score a password can receive. Anything above this score is considered a strong password.
